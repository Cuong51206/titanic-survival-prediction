## Overview
The **Titanic Survival Prediction** project is an End-to-End Machine Learning Web Application. It utilizes a Random Forest Classifier to predict the survival probability of passengers on the Titanic based on various socio-demographic features (e.g., Ticket Class, Gender, Age, and Family Size). 

A strong emphasis was placed on Exploratory Data Analysis (EDA) and Feature Engineering—such as extracting social titles from names and calculating family size—which significantly improved the model's accuracy to 82.12%.

**Live Demo:** [https://titanic-survival-prediction-13y0.onrender.com](https://titanic-survival-prediction-13y0.onrender.com)
